## 1. 项目简介

VidFlow 是一款**面向儿童的短视频浏览微信小程序**，具有以下核心特点：

| 特性 | 说明 |
|------|------|
| **目标用户** | 儿童（3-12岁）及家长 |
| **核心功能** | 视频浏览、搜索、收藏、观看历史、微信登录 |
| **设计风格** | 高级简约无边框设计，浅灰底色 `#f5f6fa`，白色卡片 + 微阴影 |
| **云服务** | 腾讯云开发（阿里云版 uniCloud） |
| **包大小** | 约 0.62MB（远低于微信 2MB 限制） |

### 功能清单

- ✅ 微信一键登录（uni-id 认证）
- ✅ 首页视频流展示（从云端数据库自动读取）
- ✅ 关键词搜索视频
- ✅ 视频播放页
- ✅ 收藏功能（云端同步）
- ✅ 观看历史记录（云端同步）
- ✅ 儿童安全过滤机制
- ✅ 无障碍支持（大字体、高对比度、简化导航）
- ✅ 使用时长提醒

## 4. 页面功能介绍

### 4.1 底部导航栏（3个主 Tab）

| Tab | 页面 | 功能 |
|-----|------|------|
| **首页** | `pages/index/index` | 展示视频列表、分类筛选、语音助手入口 |
| **搜索** | `pages/search/search` | 关键词搜索视频、搜索历史、热门推荐 |
| **我的** | `pages/user/user` | 登录/个人信息、收藏列表、观看历史 |

### 4.2 子页面（通过导航跳转）

| 页面 | 路径 | 入口方式 |
|------|------|----------|
| **视频播放** | `pages/video/video` | 点击任意视频卡片跳转 |
| **观看历史** | `pages/history/history` | "我的" → 切换到"观看历史"标签 |
| **我的收藏** | `pages/favorites/favorites` | "我的" → 切换到"我的收藏"标签 |

### 4.3 各页面详细说明

#### 首页 (index)
- 顶部：Logo + "VidFlow" 标题 + 语音助手按钮
- 搜索框：点击跳转到搜索页
- 分类标签：全部 / 动画 / 音乐 / 故事 / 教育（点击切换分类）
- 视频卡片列表：自动从云端加载视频数据

#### 搜索页 (search)
- 搜索输入框：输入关键词后按回车或点击搜索
- 搜索历史：显示最近20条搜索记录（本地存储）
- 热门推荐：动画片、儿歌、故事、画画 快捷入口
- 搜索结果：分页加载，支持"默认排序"和"最新优先"

#### 个人主页 (user)
- **未登录状态**：显示登录引导卡片，点击触发微信登录
- **已登录状态**：
  - 头像 + 昵称 + "小小探险家"徽章
  - 统计数据：观看视频数、累计时长
  - 两个标签切换：「我的收藏」和「观看历史」

#### 视频播放页 (video)
- 全屏视频播放器
- 视频信息：标题、描述、作者
- 操作按钮：点赞、收藏、分享

#### 收藏页 (favorites)
- 显示已收藏的视频列表
- 点击卡片跳转到视频播放页
- 空状态提示："还没有收藏哦~"

#### 历史页 (history)
- 显示最近观看的视频列表（按时间倒序）
- 支持清空历史记录
- 空状态提示："暂无观看历史"

## 5. 云服务架构

### 5.1 云函数列表

所有云函数位于 `uniCloud-aliyun/cloudfunctions/` 目录下：

| 函数名 | 文件路径 | 功能 | 调用方式 |
|--------|----------|------|----------|
| **auth** | `auth/index.js` | 微信登录、获取用户信息 | `callFunction('auth', {action: 'loginByWeixin', ...})` |
| **video** | `video/index.js` | 视频列表、搜索视频 | `callFunction('video', {action: 'search', keyword: 'xxx'})` |
| **favorite** | `favorite/index.js` | 收藏/取消收藏、收藏列表 | `callFunction('favorite', {action: 'add', video_id: 'xxx'})` |
| **like** | `like/index.js` | 点赞/取消点赞 | `callFunction('like', {action: 'toggle', video_id: 'xxx'})` |
| **watch-history** | `watch-history/index.js` | 观看记录、观看统计 | `callFunction('watch-history', {action: 'add', video_id: 'xxx'})` |

### 5.2 云函数通用调用模式

所有云函数都采用 **action 模式**（一个函数处理多种操作）：

```javascript
// 示例：调用 video 云函数搜索视频
const res = await callFunction('video', {
  action: 'search',     // 操作类型
  keyword: '儿歌',      // 搜索关键词
  page: 1,              // 页码
  pageSize: 10          // 每页数量
})

// 返回值统一格式
if (res.code === 0) {
  console.log(res.data.list)   // 数据列表
  console.log(res.data.total)   // 总数
}
```

## 8. 日常开发流程

### 8.1 启动开发服务器

**方法一：HBuilderX 运行（推荐）**

1. 菜单栏 → **运行** → **运行到小程序模拟器** → **微信开发者工具**
2. 或点击顶部工具栏的 ▶️ 播放按钮
3. HBuilderX 会自动编译并启动微信开发者工具

**方法二：命令行启动**

```bash
# 在项目根目录执行
npm run dev
```

然后访问 `http://localhost:5173`（仅用于 Web 调试，小程序请用方法一）

### 8.2 修改代码后的操作

1. 保存修改的文件（`Ctrl+S`）
2. HBuilderX 会**自动热重载**（Hot Reload），微信开发者工具自动刷新
3. 如果没有自动刷新，手动点击微信开发者工具的"编译"按钮

### 8.3 修改云函数后的操作

⚠️ **云函数修改后必须重新上传！**

1. 修改 `uniCloud-aliyun/cloudfunctions/xxx/index.js` 中的代码
2. 在 HBuilderX 中右键点击对应的云函数文件夹
3. 选择 **上传部署** → **上传并运行：云端安装依赖`
4. 等待上传完成（通常 10-30 秒）
5. 在小程序中测试验证

### 8.4 新增页面步骤

如果需要添加新页面：

1. **创建文件**：在 `pages/` 下新建文件夹和 `.vue` 文件
2. **注册路由**：打开 `pages.json`，在 `pages` 数组中添加：
   ```json
   {
     "path": "pages/newpage/newpage",
     "style": {
       "navigationBarTitleText": "新页面标题",
       "navigationStyle": "custom"
     }
   }
   ```
3. **编写页面**：按照现有页面的结构编写 Vue 组件

### 8.5 修改样式注意事项

当前设计规范（**请严格遵守以保持一致性**）：

## 9. 部署上线指南

### 9.1 上传代码到微信

1. HBuilderX 菜单 → **发行** → **小程序-微信（仅适用于 uni-app）**
2. 弹出的对话框中确认版本号等信息
3. 点击 **发行**
4. HBuilderX 自动编译并在微信开发者工具中打开上传界面
5. 在微信开发者工具中点击 **上传** 按钮
6. 填写版本号和备注，点击上传

### 9.2 微信公众平台审核

1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入 **管理** → **版本管理**
3. 找到刚上传的版本，点击 **提交审核**
4. 填写审核信息，等待审核（通常 1-3 个工作日）
5. 审核通过后点击 **发布**

### 9.3 上传云函数

每次修改云函数后都需要单独上传：

1. HBuilderX 中右键云函数文件夹
2. **上传部署** → **上传并运行：云端安装依赖**

> 💡 **建议**：在上传小程序代码前，先确保所有云函数都已更新部署！

### 9.4 包大小检查

微信小程序限制 **2MB**。本项目当前约 **0.62MB**，非常安全。

如遇包大小超限问题，检查以下几点：
- `static/` 目录下是否有大图片？（应 < 50KB）
- 是否有未使用的依赖？
- `unpackage/dist/` 缓存是否过大？（可以删除后重新构建）

### 12.1 如何添加新视频

视频数据存储在云数据库的 `video` 集合中。有两种添加方式：

### 12.2 视频数据模板

添加视频时必填字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `title` | String | ✅ | 视频标题，如"小兔子拔萝卜" |
| `cover_url` | String | ✅ | 封面图 URL（建议用云存储地址）|
| `video_url` | String | ✅ | 视频文件 URL |
| `description` | String | ❌ | 视频描述 |
